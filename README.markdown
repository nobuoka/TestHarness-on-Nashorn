TestHarness on Nashorn
==============================

## Demo

You can run demo tests by executing following command.

```
JAVA_HOME=/path_to/your/jdk8 PATH=$JAVA_HOME/bin:$PATH jjs js/testharness.js demo/demo_tests.js
```

JDK 8 is required.
